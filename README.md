It is a simple CRUD REST API that deals with students

To use it, You have install dependencies and have a .env file for the mongodb connection

Use the GET /students to get all students in the database

Use the GET /student/<studentid> to get a particular student

Use the POST /student to add a student

Use the PUT /student/<studentid> to change some details about a student

Use the DELETE /student/<studentid> to delete the student profile

All api are prefix with /api/v1
