It is a simple CRUD REST API that deals with students

To use it, You have install dependencies and have a .env file for the mongodb connection

Use the GET /api/v1/students to get all students in the database
Use the GET /api/v1/student/:id to get a particular student
Use the POST /api/v1/student to add a student
Use the PUT /api/v1/student/:id to change some details about a student
Use the DELETE /api/v1/student/:id to delete the student profile

And make you are connected to the local host
